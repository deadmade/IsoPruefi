Sends pending sensor data that was saved offline to MQTT broker.

Retrieves saved sensor data files from the last 24 hours, consolidates them into a single JSON payload, and publishes to a recovery topic. After successful transmission, the original files and recovery file are deleted from storage.

Parameters
mqttClientReference to the MQTT client used for publishing.
topicPrefixPrefix for the MQTT topic (e.g., "sensors/").
sensorTypeType of the sensor (e.g., "temperature").
sensorIdUnique identifier for the sensor.
nowCurrent date and time for file filtering.
